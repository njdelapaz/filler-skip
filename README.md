# Filler Skip

A Chrome extension that skips anime filler (probably)

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" in the top right corner
3. Click "Load unpacked"
4. Select this directory (`filler-skip`)
5. The extension is now installed!

## Usage

1. Navigate to crunchyroll.com
2. You'll see a "Hi! Filler Skip is active" notification
3. Click the extension icon in the toolbar to see the popup saying "Hi!"

## Files

- [manifest.json](manifest.json) - Extension configuration
- [popup.html](popup.html) - Popup interface
- [popup.js](popup.js) - Popup functionality
- [content.js](content.js) - Script that runs on Crunchyroll pages
